# 🌍 Tourism Management Platform 🇱🇰

A full-stack **MERN-based Tourism Management Platform** developed as part of the  
**IT2080 – Information Technology Project** at  
**Sri Lanka Institute of Information Technology (SLIIT)**.

This system centralizes all major travel-related services required by tourists
visiting Sri Lanka into a single, secure, and user-friendly web application.

---

## 📌 Project Overview

Tourists often face difficulties when planning trips due to fragmented platforms
for booking flights, hotels, transportation, adventure activities, and rental equipment.
This project addresses that problem by providing a **single integrated solution**
that simplifies travel planning, improves reliability, and enhances user experience.

The platform supports **multiple user roles**, **secure transactions**, **booking
management**, and **administrative control**, making it suitable for both travelers
and tourism service providers.

---

## 🎯 Objectives

- Provide a centralized tourism management system
- Reduce dependency on multiple third-party platforms
- Improve booking reliability and transparency
- Promote lesser-known tourist destinations in Sri Lanka
- Support local service providers (guides, drivers, vendors)
- Deliver a scalable, modular, and secure web application

---

## ✨ Core Features

### 👤 User Features
- User registration and authentication
- Search and browse destinations and adventure activities
- Book flights (with airline redirection)
- Reserve hotels and rooms
- Rent vehicles (self-drive or with driver)
- Purchase or rent travel accessories
- Secure online payments
- Booking history and order tracking
- Reviews and ratings

---

### 🛠️ Admin & Manager Features
- Role-based dashboards
- Destination & adventure activity management
- Vehicle, driver, and transport management
- Hotel and room management
- Flight listing and promotion management
- Accessory inventory and order management
- Booking reports and system analytics

---

## 👥 User Roles

| Role | Responsibilities |
|----|----|
| Admin | System-wide management, reports, content moderation |
| User | Browse, book, pay, and review services |
| Transport Manager | Manage vehicles, drivers, and transport bookings |
| Flight Manager | Manage flight listings and promotions |
| Hotel Owner | Manage hotels, rooms, and bookings |
| Accessory Handler | Manage accessories, inventory, and orders |
